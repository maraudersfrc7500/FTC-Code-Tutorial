*Heavy Sigh*. Alright. Guess I'm doing this now. Welcome to my FTC Tutorial! This was made shortly after the 2025 FTC Decode season ended by Robert Gillette, a former member of team 14431, the Pyrites. This tutorial covers a variety of skill levels, from beginner to advanced, aiming to help anyone that needs it. Here's a quick guide to determine where you should start:

  • Have you ever done FTC Code before in the slightest? If not, start at Beginner
  • Have you ever used FTCLib (or Solver's Lib) before? If so, start at Intermediate
  • Have you ever used more advanced coding and made autos that aren't timer based? If so, start at Advanced

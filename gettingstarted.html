Getting Started

There are a couple of things you must first do in order to code for FTC:

1. Install Android Studio

FTC Code is built in Android Studio and then sent to a device called a Control Hub (subject to change), which can be purchased by REV Robotics at their website.

Download and Follow the Installation Steps

https://developer.android.com/studio

2. Install the FtcRobotController Repository into Android Studio

Once you have Android Studio installed, download the FtcRobotController repository

https://github.com/FIRST-Tech-Challenge/FtcRobotController

After download the zip file from github, extract it and open the main folder in Android Studio

After opening it, open the TeamCode folder, the java folder, then the org.firstinspires.ftc.teamcode folder. In it, you should only see a readme.md file

Congratulations! You're officially ready to start coding FTC! If you decide to keep following along, I'll see you all soon! If not, I wish you the best of luck coding, and have fun!
